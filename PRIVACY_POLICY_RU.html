Политика конфиденциальности

Дата вступления в силу: 2024

1. Общие положения

Настоящая Политика конфиденциальности определяет порядок обработки и защиты персональных данных пользователей мобильного приложения "Строительные калькуляторы Calc1" (далее — "Приложение").

Используя Приложение, вы соглашаетесь с условиями настоящей Политики конфиденциальности.

2. Сбор информации

2.1. Данные, которые мы собираем

Приложение собирает минимально необходимую информацию:

Локальные данные (хранятся только на устройстве):

  • История расчётов (входные параметры и результаты)
  • Настройки приложения (если применимо)

Данные, которые мы НЕ собираем:

  • Персональные данные пользователя (имя, email, телефон)
  • Геолокационные данные
  • Данные устройств (IMEI, серийные номера)
  • Данные о других приложениях
  • Файлы и медиа-контент

2.2. Автоматически собираемые данные

Приложение может собирать следующие данные автоматически:

  • Версия операционной системы Android
  • Версия приложения
  • Информация об ошибках (для улучшения работы приложения)

3. Хранение данных

3.1. Локальное хранение

Все данные пользователя хранятся локально на устройстве:

  • История расчётов хранится в SharedPreferences
  • Данные не передаются на внешние серверы
  • Данные не синхронизируются между устройствами

3.2. Безопасность данных

  • Все данные хранятся в зашифрованном виде (стандартные механизмы Android)
  • Доступ к данным имеет только само приложение
  • Данные удаляются при удалении приложения

4. Использование данных

Собранные данные используются исключительно для:

  • Обеспечения работы функций приложения
  • Улучшения качества приложения
  • Исправления ошибок и багов

5. Передача данных третьим лицам

Мы НЕ передаём данные третьим лицам:

  • Данные не передаются рекламным сетям
  • Данные не передаются аналитическим сервисам
  • Данные не передаются другим приложениям
  • Данные не продаются и не сдаются в аренду

Исключения:

  • Данные могут быть переданы только по требованию законодательства
  • Данные могут быть переданы при смене владельца приложения (с уведомлением пользователей)

6. Права пользователя

Пользователь имеет право:

  • Просматривать историю своих расчётов
  • Удалять историю расчётов в любое время
  • Удалить приложение и все связанные данные
  • Запросить информацию о хранимых данных (обратившись на email)

7. Дети

Приложение предназначено для пользователей всех возрастов (3+). Мы не собираем персональные данные детей намеренно.

8. Изменения в Политике конфиденциальности

Мы оставляем за собой право вносить изменения в настоящую Политику конфиденциальности. О существенных изменениях пользователи будут уведомлены через обновление приложения.

10. Согласие

Используя Приложение, вы даёте своё согласие на обработку данных в соответствии с настоящей Политикой конфиденциальности.
