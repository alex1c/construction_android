# Строительные калькуляторы

Android приложение для расчёта строительных материалов и конструкций.

## Описание

Приложение содержит 21 калькулятор для различных строительных задач:
- Отделка и интерьер (9 калькуляторов)
- Конструкции и бетон (6 калькуляторов)
- Инженерные системы (3 калькулятора)
- Металл и электрика (3 калькулятора)

## Технологии

- Kotlin
- Jetpack Compose
- Material Design 3
- Navigation Component
- ViewModel
- StateFlow

## Требования

- Android Studio Hedgehog (2023.1.1) или новее
- JDK 17
- Android SDK 34
- Min SDK: 24 (Android 7.0)

## Запуск проекта

1. Откройте проект в Android Studio
2. Дождитесь синхронизации Gradle
3. Подключите Android устройство или запустите эмулятор
4. Нажмите Run (Shift+F10) или выберите Run > Run 'app'

## Структура проекта

```
app/src/main/java/com/construction/
├── domain/              # Доменная логика
│   ├── engine/         # Движок расчётов
│   ├── model/          # Модели данных
│   └── repository/     # Репозиторий данных
├── navigation/         # Навигация
├── ui/                 # UI компоненты
│   ├── calculator/     # Экран калькулятора
│   ├── category/      # Экран категории
│   ├── home/          # Главный экран
│   └── theme/          # Тема приложения
└── MainActivity.kt    # Главная активность
```
